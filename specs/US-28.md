# US-28 · Главный экран настроек

**Предусловие**: аутентифицирован, внутри воркспейса.

## Цель
Точка сбора пользовательских/воркспейс-настроек и отправных точек в другие стори.

## Точка входа
Сайдбар → **«Настройки»** → `/w/{id}/settings`.

## Пункты меню (in scope)
- **«Пригласить команду»** → [US-2](./US-2.md).
- **«Создать рабочее пространство»** → [US-5 · 5.1](./US-5.md).
- **«Язык»** — текущее значение `Русский`; переключение языка (details — [OQ-1](#открытые-вопросы)).
- **«Подписка»** — вне scope как отдельная стори (paywall целенаправленно drop'нут из TZ).
- **«Настройки уведомлений»** — details — [OQ-2](#открытые-вопросы).
- **«ID пользователя»** / **«ID рабочего пространства»** — отладочные, copy-to-clipboard.
- **«Выйти из аккаунта»** — logout (см. [auth.md](./auth.md)).

## Пункты, сознательно исключённые из scope
**«База знаний»**, **«Партнерская программа»**, **«Подключить CRM через Zapier»**, **«Google Календарь»**, **«API-ключи»**, **«Экспорт данных»**, **«Политика конфиденциальности»**, **«Условия использования»**, **«О нас»**, **«Присоединиться к чату сообщества»** — `drop` в `scope.json` (см. [`README.md`](./README.md)).

## Что вызывается
На бутстрэпе самого `/settings` явных tRPC-вызовов не зафиксировано — это просто меню.

Из доступных ручек (не вызывать в штатном сценарии):
- `account.deleteAccount` — по кнопке **«Удалить аккаунт»** (если она присутствует в текущем билде). Необратимая операция.
