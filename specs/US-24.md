# US-24 · Настройки создания лидов в CRM

**Предусловие**: кампания создана (см. [US-20](./US-20.md) / [US-21](./US-21.md)).

## Цель
Указать, когда кампания должна создавать записи в CRM, кто становится ответственным, и какие дефолтные property проставлять новым лидам.

## Точка входа
Внутри кампании → таб **«Настройки создания лидов в CRM»** → `/w/{id}/outreach/sequences/{sequenceId}/contact-settings`.

## Блоки

### 1. «Создавать контакт в CRM»
Radio:
- **«Когда лид ответит»**.
- **«Когда первое сообщение отправлено»**.

### 2. «Ответственные по умолчанию» (`/contact-settings/owners`)
Поиск по членам воркспейса (список из `workspaces.getMembers`), логика round-robin или по правилу (см. [OQ-1](#открытые-вопросы)).

### 3. «Значения по умолчанию для новых лидов в CRM»
Подпись: **«Укажите значения по умолчанию для новых лидов в CRM, созданных из этой кампании»**.

Для поля **«Стадия»** — dropdown воронки + кнопка **«Настроить список»**, которая открывает inline-редактор опций (**«Новое значение»**, **«Изменить цвет»**, **«Удалить»**).

## Что вызывается
- `workspaces.getMembers` — для owner-пикера.
- `outreach.sequences.patch` — сохранение всех настроек (trigger, owners, defaults).

Сигнатуры — см. [`api-contracts.md`](./api-contracts.md).

## Success-flow
- `patch` прошёл → Toast: **«Настройки создания контакта сохранены»**.

## Error-flow
`toast.error(err.message)`.

## Открытые вопросы
- **OQ-1**: логика распределения владельцев — round-robin, рандом, по правилу? Можно ли настроить в UI или зашито?
- **OQ-2**: «Настроить список» в dropdown'е «Стадии» — редактирует сам property **глобально** (см. [US-7](./US-7.md)) или делает override только в этой кампании? Для системного property ожидаем глобальное.
- **OQ-3**: какие именно property можно ставить дефолтами — только системные или любые кастомные тоже?

## Критерии приёмки
- [ ] Radio-переключатель триггера создания переключается и сохраняется.
- [ ] Owners-список загружен из `workspaces.getMembers`.
- [ ] Save → один `POST /trpc/outreach.sequences.patch`, toast показан.
- [ ] «Настроить список» для Стадии открывает редактор опций и сохраняет изменения.
